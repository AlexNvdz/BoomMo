AVISO: Para correr este proyecto es necesario tener instalado NodeJS en tu computadora.

1. instalar las dependencias del proyecto (ejecutar el comando npm install)

2. configurar la conexion a la base de datos, deberas ir al archivo env.txt y editar todas las variables que inican con DB_
si no tienes contraseña para acceder a la bd dejala en blanco
las demas variables que ya tienen un valor por defecto no las modificas

3. crear la base de datos, ve a src/database/BoomMo.sql 
 copia todo el contenido y luego ve a xampp o workbench a pegarlo en la caja de consultas
 y ejecuta el script

4. regresa a la app y ejecuta el comando npm run dev, en la terminal.

5. Entra a la direccion web arrojada por la terminal.

6. registra usuarios desde el register, para acceder como usuarios

7. para acceder al dashboard el usuario es admin y la contraseña es administrador
